Comandos Básicos do Git

No fluxo de trabalho básico do Git: você modifica arquivos no seu diretório de trabalho; você prepara os arquivos, adicionando imagens deles à sua área de preparo; você faz commit, o que leva os arquivos como eles estão na área de preparo e armazena essa imagens de forma permanente para o diretório do Git.
