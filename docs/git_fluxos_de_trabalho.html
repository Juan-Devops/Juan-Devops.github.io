Git Fluxos de Trabalho

Ao utilizar o Git para gerenciar versões de um projeto, é de suma importância definir o fluxo de trabalho (workflow), que irá estabelecer funções específicas para diferentes branches e definir quando elas devem interagir.O Git Flow trabalha com duas branches principais, a Develop e a Master, que duram para sempre; e três branches de suporte, Feature, Release e Hotfix, que são temporários e duram até realizar o merge com as branches principais
